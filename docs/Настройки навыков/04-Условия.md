# conditions — когда запретить каст

`conditions` — список проверок. Любая неуспешная проверка => отказ.

## Синтаксис

- `perm <node>`
- `lvl <min>`
- `item <MATERIAL> <AMOUNT> [where inv|hand]`
- `world allow <w1>,<w2>...`
- `world deny <w1>,<w2>...`

## Пример

```yml
conditions:
  - "perm skills.smite"
  - "lvl 10"
  - "world deny arena"
  - "item EMERALD 2 where inv"
```

Несколько миров (через запятую):

```yml
conditions:
  - "world allow world,world_nether"
```

Кулдаун задаётся отдельно: `cooldown: 5s`.
