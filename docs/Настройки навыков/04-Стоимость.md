# Cost — стоимость каста

`cost` — чем игрок платит за каст.

Зачем нужен `cost`:
- чтобы ограничить частоту и силу навыков через ресурсы (уровни опыта, предметы и т.п.).
- чтобы сделать выбор игрока осмысленным: "потратить ресурс сейчас или сохранить".

Как это работает:
- стоимость списывается только если `conditions` прошли успешно.
- если стоимость списать нельзя (например, нет предметов/уровней), каст не произойдёт.

Совет:
- стоимость лучше держать простой и очевидной, а сложную логику ("можно/нельзя") делать через `conditions`.

## Синтаксис

### `xp <levels>`

Списать указанное количество уровней опыта.

- `levels` (обязательно) — сколько уровней снять.
  - можно указывать дробные значения (например, `0.5` — половина полоски текущего уровня).

```yml
cost:
  - "xp 0.5"
```

### `<MATERIAL> [amount]`

Списать предметы указанного материала из инвентаря (включая левую руку / off-hand).

- `MATERIAL` (обязательно) — название предмета (например, EMERALD).
- `amount` (опционально) — количество.
  - если не указано — по умолчанию `1`.

```yml
cost:
  - "EMERALD 2"
```
