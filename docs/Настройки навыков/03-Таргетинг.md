# Таргетинг (`target`)

Таргетинг определяет, *на кого/куда* будут применены действия.

`target` задаётся **внутри** `groups[]`:

```yml
groups:
  - target:
      type: TARGET
      range: 5
    actions:
      - type: DAMAGE
        amount: 4
```

Поле: `target.type`

## Поддерживаемые типы

### `SELF`
Цель = кастер.

```yml
target:
  type: SELF
```

### `TARGET`
Цель = одна сущность.

- если в событии уже есть цель (например `ON_HIT`) — берётся она
- иначе плагин пытается взять сущность в прицеле (ray trace)

```yml
target:
  type: TARGET
  range: 5
```

Поля:

- `range: number` — дальность (рекомендуется задавать явно)

### `AREA`
Цели = список сущностей в радиусе.

```yml
target:
  type: AREA
  radius: 4
  center: CASTER
```

Поля:

- `radius: number` — радиус (рекомендуется задавать явно)
- `center: CASTER|TARGET` — откуда искать цели (по умолчанию `CASTER`)
