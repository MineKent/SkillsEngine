# trigger — как включить навык

`trigger` пишется **одной строкой** и определяет событие, которое запускает каст.

## Синтаксис

- `cmd <name>` — команда `/name`.
  - `<name>` — имя команды без `/`.

- `rc [MATERIAL]` — правый клик.
  - Если `[MATERIAL]` указан — триггер сработает только при клике предметом в руке этого материала.
  - Если `[MATERIAL]` не указан — сработает при любом ПКМ.

- `lc [MATERIAL]` — левый клик (аналогично `rc`).

- `hit` — срабатывает, когда игрок **нанёс удар** по сущности.

- `dmg` — срабатывает, когда игрок **получил урон**.

## Примеры

```yml
trigger: "cmd smite"
trigger: "rc BLAZE_ROD"
trigger: "lc"
trigger: "hit"
trigger: "dmg"
```
