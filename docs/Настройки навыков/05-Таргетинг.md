# Target — кого выбрать

`target` задаётся внутри `groups`.

## Синтаксис

- `self` — цель = кастер.
- `target [range]` — одна цель по взгляду с лимитом дистанции.
  - `range` опционален.
  - Для триггера `hit` можно не указывать `range` (цель уже известна из события удара).
- `area r <radius> [center caster|target]` — все цели в радиусе от точки.

## Примеры

```yml
groups:
  - target: "self"
    actions:
      - "ab self"

  - target: "target 5"
    actions:
      - "dmg 4"

  - target: "target"  # range опционален, удобно для hit

    actions:
      - "dmg 4"

  - target: "area r 4 center caster"
    actions:
      - "dmg 2"

  - target: "area r 6 center target"
    actions:
      - "particles CRIT count 30 offset 0.4"
```
