# Примеры

## Где смотреть рабочие примеры

- `plugins/SkillsEngine/skills/example.yml` — создаётся при первом запуске (если файла нет)
- исходник примера в репозитории: `src/main/resources/skills/example.yml`

## Пример (DSL): команда + условия + удар по цели

```yml
id: smite
name: "Smite"

trigger: "cmd smite"

cooldown: 2s

conditions:
  - "perm skills.smite"
  - "lvl 10"
  - "world allow world world_nether"

groups:
  - target: "target 5"
    actions:
      - "dmg 6.0"
      - "potion SLOW dur 3.0 amp 2"
      - "sound ENTITY_LIGHTNING_BOLT_THUNDER vol 0.8 pitch 1.2"
```

## Пример (DSL): клик предметом + аура по области

```yml
id: aura
trigger: "rc BLAZE_ROD"

cooldown: 10s

conditions:
  - "world deny arena"
  - "item EMERALD 1 where inv"

groups:
  - target: "area r 4 center caster"
    actions:
      - "particles CRIT count 30 offset 0.4"
      - "dmg 2.0"
      - "fire 3"
```

## Пример: смешанный формат (DSL + YAML objects)

```yml
id: mixed
trigger: "cmd mixed"

cooldown: 3s

groups:
  - target: "self"
    actions:
      - "msg \"&aHello from DSL\""
      - type: SOUND
        sound: "ENTITY_PLAYER_LEVELUP"
        volume: 1.0
        pitch: 1.0
```
