# Примеры навыков

Готовый пример есть тут:

- `plugins/SkillsEngine/skills/example.yml`
- исходник: `src/main/resources/skills/example.yml`

## Примеры

### Команда + условия + удар по цели

```yml
id: smite
trigger: "cmd smite"
cooldown: 2s

conditions:
  - "perm skills.smite"
  - "lvl 10"
  - "world allow world,world_nether"

cost:
  - "xp 1"

groups:
  - target: "target 5"
    actions:
      - "dmg 6.0"
      - "potion SLOW dur 3 amp 2"
      - "sound ENTITY_LIGHTNING_BOLT_THUNDER vol 0.8 pitch 1.2"
```

### Клик по блоку + предмет в руке через conditions + аура

```yml
id: aura
trigger: "rc STONE,DIRT"
cooldown: 10s

conditions:
  - "world deny arena"
  - "item BLAZE_ROD 1 hand"
  - "item EMERALD 1 inv"

groups:
  - target: "area 4 caster"
    actions:
      - "particles CRIT count 30 offset 0.4"
      - "dmg 2.0"
      - "fire 3"
```
