# actions — что сделать

`actions` — список действий внутри `groups`.

## Строкой (быстро)

Пример:

```yml
actions:
  - "dmg 6.0"
  - "potion SLOW dur 3 amp 2"
  - "msg \"&aHello world\""
  - "cmd console \"say cast {skill} by {player}\""
```

## Список команд

Урон/хил:
- `dmg <amount>`
- `heal <amount>`
- `fire <seconds>`

Эффекты:
- `potion <EFFECT> dur <sec> amp <n>`

Звук/частицы:
- `sound <SOUND> vol <v> pitch <p>`
- `particles <PARTICLE> count <n> offset <d> [speed <s>]`

Текст:
- `msg <text...>`
- `ab <text...>`
- `title <title> sub <subtitle...> in <t> stay <t> out <t>`

Команды:
- `cmd [console|player] <command...>`

Движение:
- `dash [power] [keepy true|false]`
- `tp <caster|target|center>`

Физика:
- `kb [strength] [y <y>]`
- `pull [strength] [y <y>]`

Прочее:
- `explosion [power] [fire true|false] [break true|false] [damage true|false]`
- `give <MATERIAL> [amount]`
- `take <MATERIAL> [amount]`
- `spawn <ENTITY> [count]`

Подсказка: если в тексте есть пробелы — берите в кавычки `"..."`.

Плейсхолдеры в тексте/командах:
- `{player}`
- `{skill}`
