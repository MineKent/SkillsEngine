## Что умеет (MVP)

- загрузка навыков из `plugins/SkillsEngine/skills/*.yml`
- триггеры: команда, клики мышью, удар по мобу, получение урона
- таргетинг: на себя / на цель / по области
- условия: permission, уровень, предметы, мир (allowed/denied)
- стоимость: XP уровни или предметы
- действия: урон/хил/эффекты/частицы/звук/команды/телепорт/и др.

## Базовая логика

1. Срабатывает `trigger` (например, команда `/fire`).
2. Плагин проверяет `conditions`, `cooldown` и `cost`.
3. Выполняются *группы действий* (`groups`). У каждой группы свой `target` и список `actions`.

### Легенда по синтаксису

- `<...>` — обязательный параметр.
- `[...]` — необязательный параметр.

Пример: `item <MAT> <AMOUNT> [where inv|hand]`.

> Важно: в текущей версии **`groups` обязательны**. Старого формата с `target`/`actions` на верхнем уровне нет.
