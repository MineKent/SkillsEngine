# Таргетинг (`target`)

Таргетинг определяет, *на кого/куда* будут применены действия.

`target` задаётся **внутри** `groups[]`.

## DSL-формат (строкой)

Пример:

```yml
groups:
  - target: "target 5"
    actions:
      - "dmg 4"
```

Поддерживаемые DSL варианты:

- `self`
- `target <range>`
- `area r <radius> [center caster|target]`

Примеры:

```yml
target: "self"
target: "target 6"
target: "area r 4 center caster"
```

## Классический YAML-формат

```yml
target:
  type: TARGET
  range: 5
```

Поддерживаемые `target.type`:

- `SELF`
- `TARGET`
- `AREA`

Поведение `TARGET`:

- если в событии уже есть цель (например `ON_HIT`) — берётся она
- иначе плагин пытается взять сущность в прицеле (ray trace)
