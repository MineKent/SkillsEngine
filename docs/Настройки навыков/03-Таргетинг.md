# target — кого выбрать

`target` задаётся внутри `groups`.

## Синтаксис

- `self` — цель = кастер (действия применяются к себе).
- `target <range>` — одна цель по взгляду, с лимитом дистанции.
- `area r <radius> [center caster|target]` — все цели в радиусе от точки.

## Пример

```yml
- target: "self"
  actions:
    - "msg hi"

- target: "target 5"
  actions:
    - "dmg 4"

- target: "area r 4 center caster"
  actions:
    - "dmg 2"

- target: "area r 6 center target"
  actions:
    - "particles CRIT count 30 offset 0.4"
```
