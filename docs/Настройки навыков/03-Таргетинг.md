# target — кого выбрать

`target` задаётся внутри `groups`.

## Синтаксис

- `self`
- `target <range>`
- `area r <radius> [center caster|target]`

## Примеры

```yml
- target: "self"
  actions:
    - "msg hi"

- target: "target 5"
  actions:
    - "dmg 4"

- target: "area r 4 center caster"
  actions:
    - "dmg 2"

- target: "area r 6 center target"
  actions:
    - "particles CRIT count 30 offset 0.4"
```
