# Cost — стоимость каста

Поле `cost` определяет, чем игрок платит за использование умения. Это список строк, где каждая строка — отдельный вид затрат.

```yml
heal-self:
  trigger: "cmd heal"
  cost:
    - "xp 1"
    - "GOLD_NUGGET 5"
  groups:
    - target: "caster"
      actions:
        - "heal 10"
```

### Как это работает

* Стоимость списывается только после успешной проверки всех `conditions`.
* Если стоимость списать нельзя (например, не хватает уровней или предметов), умение не активируется, и игрок увидит `denyMessage` (если оно задано).

> \[!TIP] Старайтесь делать стоимость простой и понятной. Сложные проверки лучше выносить в `conditions`.

## Синтаксис

### `xp <levels>`

Списывает указанное количество уровней опыта.

* `levels` (обязательно) — количество уровней. Можно использовать дробные значения (например, `0.5` — половина текущей полоски опыта).

```yml
cost:
  - "xp 0.5"
```

### `var <name> <amount>`

Списывает значение переменной при касте.

```yml
cost:
  - "var souls 5"
```

### `mana <amount>`

Списывает ману (если у вас есть плагин на ману, который поддерживает SkillsEngine).

* `amount` (обязательно) — количество маны.

```yml
cost:
  - "mana 10"
```

### `<MATERIAL> [amount]`

Списывает предметы указанного типа из инвентаря игрока.

* `MATERIAL` (обязательно) — название материала (например, `EMERALD`).
* `amount` (опционально) — количество. По умолчанию `1`.

```yml
cost:
  - "EMERALD 2"
```
