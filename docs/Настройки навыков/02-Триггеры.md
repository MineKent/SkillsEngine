# trigger — как включить навык

Самый простой способ — строкой.

## Строкой

- `cmd <name>` — команда `/name`
- `rc [MATERIAL]` — ПКМ (материал можно не указывать)
- `lc [MATERIAL]` — ЛКМ (материал можно не указывать)
- `hit` — когда игрок ударил
- `dmg` — когда игрок получил урон

Примеры:

```yml
trigger: "cmd smite"
trigger: "rc BLAZE_ROD"
trigger: "hit"
```

## Объектом (если надо)

```yml
trigger:
  type: COMMAND
  command: "smite"
```
