# Примеры умений

В одном файле можно определять несколько умений. Каждое умение — это ключ верхнего уровня.

```yml
# Умение "Удар молнии"
smite:
  # Активация командой /smite
  trigger: "cmd smite"
  cooldown: 2s
  conditions:
    - "perm skills.smite"
    - "lvl 10"
    - "world allow world,world_nether"
  cost:
    - "xp 1"
  groups:
    - target: "target 5" # Цель, на которую смотрит игрок в радиусе 5 блоков
      actions:
        - "dmg 6.0"
        - "potion SLOW dur 3 amp 2"
        - "sound ENTITY_LIGHTNING_BOLT_THUNDER vol 0.8 pitch 1.2"

# Умение "Аура огня"
aura:
  # Активация по правому клику на камень или землю
  trigger: "rc block STONE,DIRT"
  cooldown: 10s
  conditions:
    # Запрещено в мире 'arena'
    - "world deny arena" 
    # Требует Жезл Ифрита в руке
    - "item BLAZE_ROD 1 hand" 
    # Требует изумруд в инвентаре
    - "item EMERALD 1 inv" 
  groups:
    - target: "area 4 caster" # Цели в радиусе 4 блоков вокруг кастера
      actions:
        - "particles CRIT count 30 offset 0.4"
        - "dmg 2.0"
        - "fire 3"
```
