# Trigger — Как включить навык

`trigger` — одна строка.

## Синтаксис

* `cmd <name>` — каст при вводе команды `/name`.
* `rc [MATERIAL]` — каст при ПКМ, опционально по предмету в руке.
* `lc [MATERIAL]` — каст при ЛКМ, опционально по предмету в руке.
* `hit` — каст когда игрок наносит удар по сущности.
* `dmg` — каст когда игрок получает урон.

## Пример

```yml
trigger: "cmd smite"
trigger: "rc BLAZE_ROD"
trigger: "lc"
trigger: "hit"
trigger: "dmg"
```
