# Trigger — как включить навык

`trigger` — одна строка (DSL), которая определяет событие, при котором навык будет кастоваться.

## Синтаксис

### `cmd <name>`

Каст при вводе команды `/name`.

- `name` (обязательно) — имя команды без `/`.

### `rc [BLOCKS]`

Каст при ПКМ.

- `BLOCKS` (опционально) — Material блока (или список через запятую), по которому должен кликнуть игрок.
  - пример одного: `STONE`
  - пример списка: `STONE,DIRT,OAK_LOG`
  - если `BLOCKS` указан, то клик по воздуху **не** сработает.

Важно:
- Проверку предмета в руке делайте через `conditions`, например: `item BLAZE_ROD 1 hand`

### `lc [BLOCKS]`

Каст при ЛКМ.

- `BLOCKS` (опционально) — Material блока (или список через запятую), по которому должен кликнуть игрок.
  - пример одного: `STONE`
  - пример списка: `STONE,DIRT,OAK_LOG`
  - если `BLOCKS` указан, то клик по воздуху **не** сработает.

### `hit [MOBS]`

Каст когда игрок наносит урон.

- `MOBS` (опционально) — EntityType или список EntityType через запятую.
  - пример: `ZOMBIE` или `ZOMBIE,SKELETON`

### `dmg [MOBS]`

Каст когда игрок получает урон.

- `MOBS` (опционально) — EntityType или список EntityType через запятую.
  - пример: `ZOMBIE` или `ZOMBIE,SKELETON`

## Примеры

Короткий список: 1 строка на каждый метод.

```yml
# cmd
trigger: "cmd smite"

# rc / lc
trigger: "rc"
trigger: "rc STONE,DIRT"
trigger: "lc"
trigger: "lc STONE"

# hit / dmg
trigger: "hit"
trigger: "hit ZOMBIE,SKELETON"
trigger: "dmg"
trigger: "dmg ZOMBIE"
```
