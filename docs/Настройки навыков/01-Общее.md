# Навык — общий вид

Файлы навыков: `plugins/SkillsEngine/skills/*.yml`

- 1 файл = 1 навык
- если `id` не задан — берётся имя файла
- `groups` обязательно (хотя бы 1 группа)

## Примеры

```yml
id: fire
trigger: "cmd fire"
cooldown: 5s

groups:
  - target: "target 5"
    actions:
      - "fire 3"
```

## Синтаксис

### Легенда

- `<...>` — обязательно
- `[...]` — опционально

Пример: `item <MATERIAL> <AMOUNT> [inv|hand|mainhand|offhand]`.
Пример со значениями: `item EMERALD 1 mainhand`.

### Поля навыка

- `trigger` — как включить навык.
- `cooldown` — задержка между кастами.
- `conditions` — когда запретить каст.
- `cost` — чем платить.
- `denyMessage` — что писать игроку при отказе.
- `groups` — что делать и по каким целям.

### `groups`

Каждая группа содержит:
- `target` — кого выбрать
- `actions` — что сделать

Важно:
- Если `target` группы не нашёл цели, то группа пропускается.

Пример:

```yml
groups:
  - target: "caster"
    actions:
      - "msg &aHello"
      - "sound ENTITY_PLAYER_LEVELUP vol 1 pitch 1"
```
