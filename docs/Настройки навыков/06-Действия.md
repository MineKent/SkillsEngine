# actions — что сделать

`actions` — **список строк** внутри `groups`. Каждая строка — одна команда движка.

## Быстрый пример

```yml
actions:
  - "dmg 6.0"
  - "potion SLOW dur 3 amp 2"
  - "sound ENTITY_PLAYER_LEVELUP vol 1.0 pitch 1.0"
  - "msg &aHello world"
  - "cmd console say cast {skill} by {player}"
```

## Команды

Урон/лечение:
- `dmg <amount>`
- `heal <amount>`
- `fire <seconds>`

Эффекты:
- `potion <EFFECT> dur <sec> amp <n>`

Звук/частицы:
- `sound <SOUND> vol <v> pitch <p> [mode <WORLD|CASTER|TARGETS>] [at <CASTER|TARGET|CENTER>]`
- `particles <PARTICLE> [count <n>] [offset <d>] [speed <s>] [at <CASTER|TARGET|CENTER>]`

Текст:
- `msg <text...>`
- `ab <text...>`
- `title <title> sub <subtitle...> in <t> stay <t> out <t>`

Команды:
- `cmd [console|player] <command...>`

Движение:
- `dash [power] [keepy true|false]`
- `tp <caster|target|center>`

Физика:
- `kb [strength] [y <y>]`
- `pull [strength] [y <y>]`

Прочее:
- `explosion [power] [fire true|false] [break true|false] [damage true|false] [at <CASTER|TARGET|CENTER>]`
- `give <MATERIAL> [amount]`
- `take <MATERIAL> [amount]`
- `spawn <ENTITY> [count]`

## Плейсхолдеры

- `{player}`
- `{skill}`
