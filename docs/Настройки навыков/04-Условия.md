# conditions — когда запретить каст

`conditions` — список строк (или можно объектами).

## Строкой

- `perm <node>` — нужно право
- `lvl <min>` — нужен уровень
- `item <MAT> <AMOUNT> [where inv|hand]` — нужен предмет
- `world allow <w1> <w2>...` — только эти миры
- `world deny <w1> <w2>...` — запретить в этих мирах

Пример:

```yml
conditions:
  - "perm skills.smite"
  - "lvl 10"
  - "world deny arena"
  - "item EMERALD 2 where inv"
```

Кулдаун задаётся отдельно: `cooldown: 5s`.
