# Cost — стоимость каста

`cost` — чем игрок платит за каст.

## Синтаксис

```yml
cost:
  - "xp <levels>"
  - "<MATERIAL> [amount]"
```

Где:
- `xp <levels>` — списать указанное количество уровней опыта.
- `<MATERIAL> [amount]` — списать предметы указанного материала из инвентаря.
  - `amount` опционален, по умолчанию `1`.

## Примеры

Короткий список: 1 строка на каждый метод.

```yml
cost:
  - "xp 1"
  - "EMERALD 2"
  - "DIAMOND"   # amount по умолчанию 1
```
