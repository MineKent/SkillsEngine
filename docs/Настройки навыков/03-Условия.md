# Conditions — когда запретить каст

`conditions` — список проверок. Любая неуспешная проверка => отказ.

## Синтаксис

- `perm <node>` — требовать permission.
- `lvl <min>` — требовать уровень игрока не ниже `<min>`.
- `item <MATERIAL> [AMOUNT] [inv|hand|mainhand|offhand]` — требовать предмет в указанном месте.
  - если `AMOUNT` не указан — по умолчанию `1`
  - `inv` — инвентарь
  - `hand` — любая рука (основная или вторая)
  - `mainhand` — основная рука
  - `offhand` — вторая рука
- `world allow <w1>,<w2>...` — разрешить каст только в указанных мирах.
- `world deny <w1>,<w2>...` — запретить каст в указанных мирах.

## Примеры

```yml
conditions:
  - "perm skills.smite"
  - "lvl 10"
  - "item EMERALD 2 inv"
  - "item SHIELD offhand"
  - "world allow world,world_nether"
  - "world deny arena"
```
