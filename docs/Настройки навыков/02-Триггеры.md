# Триггеры (`trigger`)

Триггер определяет, *когда* навык пытается кастоваться.

Поле: `trigger.type`

## Поддерживаемые типы

### `COMMAND`
Срабатывает по команде игрока.

```yml
trigger:
  type: COMMAND
  command: "fire"
```

Поля:

- `command: string` — команда без `/` (обязательно)

### `RIGHT_CLICK` / `LEFT_CLICK`
Срабатывает при клике (по воздуху/блоку).

```yml
trigger:
  type: RIGHT_CLICK
  material: "DIAMOND_SWORD"
```

Поля:

- `material: string` — опционально. Если указано, триггер сработает только если предмет в **main hand** совпадает.

### `ON_HIT`
Срабатывает, когда игрок **ударил** сущность.

```yml
trigger:
  type: ON_HIT
```

### `ON_DAMAGE`
Срабатывает, когда игрок **получил** урон.

```yml
trigger:
  type: ON_DAMAGE
```
