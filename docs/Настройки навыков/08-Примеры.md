# Примеры умений

В одном файле можно определять несколько умений. Каждое умение — это ключ верхнего уровня.

```yml
# Умение "Удар молнии"
smite:
  # Активация командой /smite
  trigger: "cmd smite"
  cooldown: 2s
  conditions:
    - "perm skills.smite"
    - "lvl 10"
    - "world allow world,world_nether"
  cost:
    - "xp 1"
  groups:
    - target: "target 5" # Цель, на которую смотрит игрок в радиусе 5 блоков
      actions:
        - "dmg 6.0"
        - "potion SLOW dur 3 amp 2"
        - "sound ENTITY_LIGHTNING_BOLT_THUNDER vol 0.8 pitch 1.2"

# Умение "Аура огня"
aura:
  # Активация по правому клику на камень или землю
  trigger: "rc block STONE,DIRT"
  cooldown: 10s
  conditions:
    # Запрещено в мире 'arena'
    - "world deny arena" 
    # Требует Жезл Ифрита в руке
    - "item BLAZE_ROD 1 hand" 
    # Требует изумруд в инвентаре
    - "item EMERALD 1 inv" 
  groups:
    - target: "area 4 caster" # Цели в радиусе 4 блоков вокруг кастера
      actions:
        - "particles CRIT count 30 offset 0.4"
        - "dmg 2.0"
        - "fire 3"
```

## Пример (акцент на действия)

Небольшое комбо из нескольких эффектов: сообщение, actionbar, частицы, звук, задержка и урон.

```yml
combo_strike:
  trigger: "cmd combo"
  cooldown: 6s
  conditions:
    - "health >30%"
  groups:
    - target: "target 6"
      actions:
        - "msg &eВы используете &6Комбо-удар&e!"
        - "ab &aЦель поражена!"
        - "particles SWEEP_ATTACK count 20 offset 0.3"
        - "sound ENTITY_PLAYER_ATTACK_SWEEP vol 0.8 pitch 1.2"
        - "wait 10"
        - "dmg 8"
        - "sound ENTITY_PLAYER_ATTACK_STRONG vol 0.9 pitch 0.9"
```

## Пример (акцент на таргетинг)

AoE-умение от точки взгляда, с исключением кастера и фильтрацией целей.

```yml
shockwave:
  trigger: "cmd shockwave"
  cooldown: 12s
  groups:
    # Центр области — точка по лучу взгляда (до 20 блоков)
    # Флаги: air,skipcaster
    # Фильтр: не бить игроков
    - target: "area 6 look 20 air,skipcaster filter(deny player)"
      actions:
        - "particles EXPLOSION_NORMAL count 60 offset 0.8"
        - "sound ENTITY_GENERIC_EXPLODE vol 0.6 pitch 1.0"
        - "dmg 5"
```

## Пример (акцент на переменные)

Система "ярости": копим переменную, а при достижении порога 50 используем усиленную атаку и списываем ресурс через cost.

```yml
rage_hit:
  trigger: "hit"
  cooldown: 0.2s
  groups:
    - target: "target"
      actions:
        - "var edit rage + 5"
        - "ab &cЯрость: {var:rage}"

rage_burst:
  trigger: "cmd rage"
  cooldown: 8s
  conditions:
    - "var rage >= 50"
  cost:
    - "var rage 50"
  groups:
    - target: "area 4 caster skipcaster"
      actions:
        - "msg &cВЫПЛЕСК ЯРОСТИ!"
        - "particles CRIT count 80 offset 0.7"
        - "sound ENTITY_WITHER_SPAWN vol 0.4 pitch 1.5"
        - "dmg {math:10 + {var:rage} / 10}"
```

## Пример (акцент на условиях и стоимости)

Навык доступен только ночью и только в определённых биомах, а также требует ресурс (опыт + переменная).

```yml
night_hunt:
  trigger: "cmd hunt"
  cooldown: 15s
  conditions:
    - "time night"
    - "biome allow FOREST,DARK_FOREST"
    - "health >40%"
    - "var souls >= 5"
  cost:
    - "xp 1"
    - "var souls 5"
  groups:
    - target: "target 8 filter(allow monster)"
      actions:
        - "msg &5Вы чувствуете зов ночи..."
        - "potion SPEED dur 6 amp 1"
        - "particles PORTAL count 40 offset 0.6"
        - "sound ENTITY_ENDERMAN_AMBIENT vol 0.5 pitch 1.4"
        - "dmg 7"
```

## Пример (акцент на плейсхолдерах и математике)

Показывает, как делать вычисления через `{math:...}` и использовать `{var:...}` в тексте и числах.

```yml
stats_demo:
  trigger: "cmd stats"
  cooldown: 1s
  groups:
    - target: "caster"
      actions:
        - "var set damage_dealt = {math:{var:damage_dealt} + 1}"
        - "msg &eСчётчик: &6{var:damage_dealt}"
        - "msg &eФормула урона: &6{math:10 + {var:damage_dealt} / 2}"
```
