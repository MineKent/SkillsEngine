# Trigger — как включить навык

`trigger` — одна строка (DSL), которая определяет событие, при котором навык будет кастоваться.

## Синтаксис

### `cmd`

- `cmd <name>` — каст при вводе команды `/name`.

### `rc` / `lc` (клики мышью)

- `rc [BLOCKS]` — каст при ПКМ.
- `lc [BLOCKS]` — каст при ЛКМ.

Где:
- `BLOCKS` — по какому блоку (или списку блоков через запятую) кликает игрок (опционально).
  - Пример одного: `STONE`
  - Пример списка: `STONE,DIRT,OAK_LOG`
  - Если `BLOCKS` указан, то клик по воздуху **не** сработает.

Важно:
- Проверку предмета в руке делайте через `conditions`, например: `item BLAZE_ROD 1 hand`

### `hit` / `dmg` (урон)

- `hit [MOBS]` — каст когда игрок наносит урон.
- `dmg [MOBS]` — каст когда игрок получает урон.

Где:
- `MOBS` — EntityType или список EntityType через запятую. Опционально.
  - Пример: `ZOMBIE` или `ZOMBIE,SKELETON`

## Примеры

Короткий список: 1 строка на каждый метод.

```yml
# cmd
trigger: "cmd smite"

# rc / lc
trigger: "rc"
trigger: "rc STONE,DIRT"
trigger: "lc"
trigger: "lc STONE"

# hit / dmg
trigger: "hit"
trigger: "hit ZOMBIE,SKELETON"
trigger: "dmg"
trigger: "dmg ZOMBIE"
```
