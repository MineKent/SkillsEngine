# Триггеры (`trigger`)

Триггер определяет, *когда* навык пытается кастоваться.

`trigger` можно задавать двумя способами:

- объектом YAML (`trigger.type`, `trigger.command`, ...)
- строкой DSL (сокращённо)

## DSL (рекомендуется для простых навыков)

```yml
trigger: "cmd fire"
```

Поддерживаемые DSL варианты:

- `cmd <name>` — команда игрока (`/name`)
- `rc [MATERIAL]` — right click (материал опционален)
- `lc [MATERIAL]` — left click (материал опционален)
- `hit` — когда игрок ударил сущность
- `dmg` — когда игрок получил урон

Примеры:

```yml
trigger: "cmd smite"
trigger: "rc BLAZE_ROD"
trigger: "hit"
```

## Классический YAML-формат

### `COMMAND`

```yml
trigger:
  type: COMMAND
  command: "fire"
```

### `RIGHT_CLICK` / `LEFT_CLICK`

```yml
trigger:
  type: RIGHT_CLICK
  material: "DIAMOND_SWORD"
```

### `ON_HIT`

```yml
trigger:
  type: ON_HIT
```

### `ON_DAMAGE`

```yml
trigger:
  type: ON_DAMAGE
```
