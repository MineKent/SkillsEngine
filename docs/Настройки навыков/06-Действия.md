# Actions — что сделать

`actions` — список строк внутри `groups`. Каждая строка — одна команда движка.

## Синтаксис

- `dmg <amount>` — нанести урон каждой цели из текущего `target`.
- `heal <amount>` — вылечить каждую цель.
- `fire <seconds>` — поджечь цель.
- `potion <EFFECT> dur <sec> amp <n>` — выдать эффект.
- `sound <SOUND> vol <v> pitch <p> [mode <WORLD|CASTER|TARGETS>] [at <CASTER|TARGET|CENTER>]` — проиграть звук.
- `particles <PARTICLE> [count <n>] [offset <d>] [speed <s>] [at <CASTER|TARGET|CENTER>]` — частицы.
- `msg <text...>` — сообщение в чат кастеру.
- `ab <text...>` — actionbar кастеру.
- `title <title> sub <subtitle...> in <t> stay <t> out <t>` — title/subtitle.
- `cmd [console|player] <command...>` — выполнить команду.
- `dash [power] [keepy true|false]` — рывок.
- `tp <caster|target|center>` — телепорт.
- `kb [strength] [y <y>]` — оттолкнуть.
- `pull [strength] [y <y>]` — притянуть.
- `explosion [power] [fire true|false] [break true|false] [damage true|false] [at <CASTER|TARGET|CENTER>]` — взрыв.
- `give <MATERIAL> [amount]` — выдать предмет цели.
- `take <MATERIAL> [amount]` — забрать предмет у цели.
- `spawn <ENTITY> [count]` — заспавнить сущность у цели.

## Примеры

```yml
groups:
  - target: "self"
    actions:
      - "dmg 6.0"
      - "potion SLOW dur 3 amp 2"
      - "sound ENTITY_PLAYER_LEVELUP vol 1.0 pitch 1.0"
      - "msg &aHello world"
      - "cmd console say cast {skill} by {player}"
```

## Плейсхолдеры

- `{player}`
- `{skill}`
