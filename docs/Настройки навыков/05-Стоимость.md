# Стоимость (`cost`)

Стоимость определяет, *чем игрок платит* за каст.

Формат:

```yml
cost:
  xpLevels: 1
```

или

```yml
cost:
  material: "EMERALD"
  amount: 2
```

Если ресурсов не хватает — каст отклоняется.

## Варианты

### XP уровни

- `xpLevels: number` — списать указанное количество уровней

### Предмет

- `material: string` — материал Bukkit (например `EMERALD`).
- `amount: number` — количество (по умолчанию 1).

> Важно: стоимость списывается только если каст не отклонён по условиям/кулдауну и есть цель (если требуется таргетинг).
