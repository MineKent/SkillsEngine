# target — кого выбрать

`target` пишется внутри `groups`.

## Строкой

- `self` — сам игрок
- `target <range>` — одна цель
- `area r <radius> [center caster|target]` — все в радиусе

Примеры:

```yml
- target: "self"
  actions:
    - "msg \"hi\""

- target: "target 5"
  actions:
    - "dmg 4"

- target: "area r 4 center caster"
  actions:
    - "dmg 2"
```
