# Действия (`actions`)

`actions` выполняются **внутри группы** (`groups[]`).

Каждый элемент `actions` может быть:

- объектом YAML с `type: ...`
- строкой DSL

## DSL-формат (быстрые действия)

Пример:

```yml
actions:
  - "dmg 6.0"
  - "potion SLOW dur 3.0 amp 2"
  - "sound ENTITY_PLAYER_LEVELUP vol 1.0 pitch 1.0"
  - "msg \"&aHello world\""
  - "cmd console \"say Cast by {player}\""
```

### Список поддерживаемых DSL actions

Базовые:

- `dmg <amount>`
- `heal <amount>`
- `fire <seconds>`

Эффекты:

- `potion <EFFECT> dur <sec> amp <n>`
  - `dur` поддерживает дробные секунды (внутри преобразуется в `durationTicks`)

Звук/частицы:

- `sound <SOUND> vol <v> pitch <p> [mode <WORLD|CASTER|TARGETS>] [at <CASTER|TARGET|CENTER>]`
- `particles <PARTICLE> [count <n>] [offset <d>] [speed <s>] [at <CASTER|TARGET|CENTER>]`

Сообщения:

- `msg <text...>` — чат (для пробелов используйте кавычки)
- `ab <text...>` — actionbar
- `title <title> [sub <subtitle...>] [in <t>] [stay <t>] [out <t>]`

Команды:

- `cmd [console|player] <command...>`

Перемещение:

- `dash [power] [keepy true|false]`
- `tp <caster|target|center>`

Физика:

- `kb [strength] [y <y>]`
- `pull [strength] [y <y>]`

Прочее:

- `explosion [power] [fire true|false] [break true|false] [damage true|false] [at <CASTER|TARGET|CENTER>]`
- `give <MATERIAL> [amount]`
- `take <MATERIAL> [amount]`
- `spawn <ENTITY> [count]`

## Классический YAML-формат

Всё ещё поддерживается:

```yml
actions:
  - type: DAMAGE
    amount: 4
  - type: POTION
    effect: SPEED
    duration: 3
    amplifier: 1
```

## Плейсхолдеры в тексте/командах

В `msg/ab/title/cmd` можно использовать:

- `{player}` — ник кастера
- `{skill}` — id навыка
