# Cost - Стоимость каста

Стоимость — чем игрок платит за каст.

## Синтаксис

- `xpLevels: <n>` — списать `<n>` уровней опыта у игрока.
- `material: <MATERIAL>` — списать предмет указанного материала из инвентаря.
- `amount: <AMOUNT>` — количество предметов для списания (по умолчанию 1).

## Пример

XP:

```yml
cost:
  xpLevels: 1
```

Предмет:

```yml
cost:
  material: "EMERALD"
  amount: 2
```
