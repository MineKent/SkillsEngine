# Таргетинг (target)

`target.type` поддерживает:

- `SELF` — цель = кастер
- `TARGET` — цель = entity
  - поля: `range: number` (если цель не пришла из события, берётся entity в прицеле)
- `AREA` — список целей в радиусе
  - поля: `radius: number`
  - поля (опц.): `center: CASTER|TARGET` (по умолчанию CASTER)
