# Cost — стоимость каста

`cost` — чем игрок платит за каст.

## Синтаксис

### `xp <levels>`

Списать указанное количество уровней опыта.

- `levels` (обязательно) — сколько уровней снять.

### `<MATERIAL> [amount]`

Списать предметы указанного материала из инвентаря.

- `MATERIAL` (обязательно) — Material предмета.
- `amount` (опционально) — количество.
  - если не указано — по умолчанию `1`.

## Примеры

Короткий список: 1 строка на каждый метод.

```yml
cost:
  - "xp 1"
  - "EMERALD 2"
  - "DIAMOND"   # amount по умолчанию 1
```
