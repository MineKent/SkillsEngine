# Навык: общий вид

Файлы навыков: `plugins/SkillsEngine/skills/*.yml`

- 1 файл = 1 навык
- если `id` не задан — берётся имя файла
- `groups` обязательно (хотя бы 1 группа)

## Быстрый пример

```yml
id: fire
trigger: "cmd fire"
cooldown: 5s

groups:
  - target: "target 5"
    actions:
      - "fire 3"
```

## Что можно писать в навыке

- `trigger` — как сработает навык
- `cooldown` — задержка между кастами
- `conditions` — когда каст запрещать
- `cost` — чем платить
- `denyMessage` — что писать игроку при отказе
- `groups` — что делать и по каким целям

## `groups` — группы действий

Каждая группа:

- `target` — кого выбрать
- `actions` — что сделать

Пример:

```yml
groups:
  - target: "self"
    actions:
      - "msg \"&aHello\""
      - "sound ENTITY_PLAYER_LEVELUP vol 1 pitch 1"
```
