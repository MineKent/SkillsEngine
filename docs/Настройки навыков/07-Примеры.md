# Примеры

## Где смотреть рабочие примеры

- `plugins/SkillsEngine/skills/example.yml` — создаётся при первом запуске (если файла нет)
- исходник примера в репозитории: `src/main/resources/skills/example.yml`

## Пример: удар по цели с эффектами

```yml
id: smite
name: "Smite"

trigger:
  type: ON_HIT

cooldown: 2s

groups:
  - target:
      type: TARGET
      range: 5
    actions:
      - type: DAMAGE
        amount: 6
      - type: POTION
        effect: "SLOW"
        duration: 3
        amplifier: 0
      - type: SOUND
        sound: "ENTITY_LIGHTNING_BOLT_THUNDER"
        volume: 0.8
        pitch: 1.2
```

## Пример: аура вокруг игрока

```yml
id: aura
name: "Aura"

trigger:
  type: RIGHT_CLICK
  material: "BLAZE_ROD"

cooldown: 10s

groups:
  - target:
      type: AREA
      radius: 4
      center: CASTER
    actions:
      - type: DAMAGE
        amount: 2
      - type: PARTICLES
        particle: "CRIT"
        count: 30
        offset: 0.4
```
