# Cost — Стоимость каста

`cost` — чем игрок платит за каст.

Поддерживаются **2 формата**:

1) **Новый (рекомендуемый)**: список строк
2) Legacy: объект с `xpLevels/material/amount`

## Новый формат (список строк)

```yml
cost:
  - "xp 1"
  - "EMERALD 2"
```

Правила:
- Каждая строка — один элемент стоимости.
- `xp <n>` — списать `<n>` уровней опыта.
- `<MATERIAL> <amount>` — списать предметы указанного материала из инвентаря.
  - `amount` опционален, по умолчанию `1`.

Примеры:

```yml
cost:
  - "xp 3"
```

```yml
cost:
  - "DIAMOND 1"
  - "EMERALD 2"
```

## Legacy формат (объект)

XP:

```yml
cost:
  xpLevels: 1
```

Предмет:

```yml
cost:
  material: "EMERALD"
  amount: 2
```
