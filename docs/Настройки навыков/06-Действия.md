# Действия (`actions`)

`actions` — список объектов, где у каждого есть поле `type`.

Действия выполняются **внутри группы** (`groups[]`).

```yml
groups:
  - target:
      type: TARGET
      range: 5
    actions:
      - type: DAMAGE
        amount: 4
      - type: SOUND
        sound: "ENTITY_PLAYER_LEVELUP"
```

Ниже перечислены поддерживаемые действия и их основные параметры.

## Бой / эффекты

### `DAMAGE`
Нанести урон всем целям.

- `amount: number` (например `4.0`)

### `HEAL`
Лечить цели.

- `amount: number` (например `6.0`)

### `POTION`
Дать эффект.

- `effect: string` — например `SPEED` (обязательно)
- `duration: number` — секунды (или `durationTicks`)
- `amplifier: number`
- `ambient: bool`
- `particles: bool`
- `icon: bool`

### `SET_FIRE`
Поджечь цели.

- `seconds: number` (или `ticks`)

## Перемещение

### `DASH`
Рывок кастера вперёд.

- `power: number`
- `keepY: bool`

### `TELEPORT`
Телепорт кастера.

- `to: CASTER|TARGET|CENTER|LOCATION`
- для `LOCATION`: `world`, `x`, `y`, `z`, (опционально) `yaw`, `pitch`

### `KNOCKBACK`
Оттолкнуть цели от кастера.

- `strength: number`
- `y: number`

### `PULL`
Притянуть цели к кастеру.

- `strength: number`
- `y: number`

## Визуал / звук

### `PARTICLES`
Спавн частиц.

- `particle: string` — например `CRIT`
- `count: number`
- `at: CASTER|TARGET|CENTER` (опц.)
- `offset` или `offsetX/offsetY/offsetZ`
- `speed`
- `show_all_players: bool` (Paper)

Для `DUST`/`DUST_COLOR_TRANSITION`:

- `color: "#ff0000"` или `"255,0,0"`
- `toColor: "#ffffff"` (только для transition)
- `size: number`

### `SOUND`
Проиграть звук.

- `sound: string` — например `ENTITY_PLAYER_LEVELUP`
- `volume: number`
- `pitch: number`
- `category: string` (опц.)
- `mode: WORLD|CASTER|TARGETS`
- `at: CASTER|TARGET|CENTER` (опц.)

## Сообщения / команды / предметы

### `MESSAGE`
Сообщение кастеру.

- `text: string` — поддерживает `&` цвета
- `mode: CHAT|ACTIONBAR|TITLE` (опц.)

### `ACTIONBAR`
- `text: string`

### `TITLE`
- `title: string`
- `subtitle: string`
- `fadeIn: number`
- `stay: number`
- `fadeOut: number`

### `COMMAND`
Выполнить команду.

- `executor: CONSOLE|PLAYER` (по умолчанию `CONSOLE`)
- `command: string` — можно использовать плейсхолдеры, см. ниже

### `GIVE_ITEM`
Выдать предмет кастеру.

- `material: string`
- `amount: number`

### `TAKE_ITEM`
Забрать предмет у кастера.

- `material: string`
- `amount: number`

### `SPAWN_ENTITY`
Заспавнить сущность.

- `entity: string` — например `ZOMBIE`
- `count: number`

## Плейсхолдеры в тексте/командах

В `MESSAGE`, `ACTIONBAR`, `TITLE`, `COMMAND` можно использовать:

- `{player}` — ник кастера
- `{skill}` — id навыка
