# Trigger — как включить навык

`trigger` — одна строка (DSL), которая определяет событие, при котором навык будет кастоваться.

## Синтаксис

### `cmd`

- `cmd <name>` — каст при вводе команды `/name`.

### `rc` / `lc` (клики мышью)

- `rc [BLOCKS]` — каст при ПКМ.
- `lc [BLOCKS]` — каст при ЛКМ.

Где:
- `BLOCKS` — по какому блоку (или списку блоков через запятую) кликает игрок (опционально).
  - Пример одного: `STONE`
  - Пример списка: `STONE,DIRT,OAK_LOG`
  - Если `BLOCKS` указан, то клик по воздуху **не** сработает.

Важно:
- Проверку предмета в руке делайте через `conditions`, например:

```yml
conditions:
  - "item BLAZE_ROD 1 hand"
```

Поддержанные формы записи:
- `rc`
- `rc STONE`
- `rc STONE,DIRT`

### `hit` / `dmg` (урон)

- `hit [MOBS]` — каст когда игрок **наносит урон** (бьёт кого-то).
  - `MOBS` — список EntityType через запятую, по которым игрок ударил. Опционально.

- `dmg [MOBS]` — каст когда игрок **получает урон**.
  - `MOBS` — список EntityType через запятую, которые нанесли урон игроку. Опционально.

Пример `MOBS`: `ZOMBIE` или `ZOMBIE,SKELETON`.

## Примеры

```yml
trigger: "cmd smite"

trigger: "rc"                          # любой ПКМ
trigger: "rc STONE,DIRT"                # ПКМ по камню/земле
# предмет в руке проверяйте через conditions:
# conditions:
#   - "item BLAZE_ROD 1 hand"

trigger: "lc"                          # любой ЛКМ

trigger: "hit"                         # удар по любой сущности
trigger: "hit ZOMBIE,SKELETON"         # удар только по ZOMBIE или SKELETON

trigger: "dmg"                         # получение урона от любого источника
trigger: "dmg ZOMBIE"                  # получение урона только от ZOMBIE
```
