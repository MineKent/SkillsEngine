# Cost — Стоимость каста

`cost` — чем игрок платит за каст.

## Формат

`cost` задаётся **списком строк**:

```yml
cost:
  - "xp 1"
  - "EMERALD 2"
```

## Правила

- Каждая строка — один элемент стоимости.
- `xp <n>` — списать `<n>` уровней опыта.
- `<MATERIAL> [amount]` — списать предметы указанного материала из инвентаря.
  - `amount` опционален, по умолчанию `1`.

## Примеры

Только опыт:

```yml
cost:
  - "xp 3"
```

Только предметы:

```yml
cost:
  - "DIAMOND 1"
  - "EMERALD 2"
```

Опыт + предметы:

```yml
cost:
  - "xp 5"
  - "GOLD_INGOT 16"
```
