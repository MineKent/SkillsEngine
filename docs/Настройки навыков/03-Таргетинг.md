# target — кого выбрать

`target` задаётся **внутри `groups`** и определяет, по каким сущностям будут выполнены `actions` этой группы.

## Синтаксис

- `self` — цель = сам кастер (игрок, который активировал навык).

- `target <range>` — цель = одна сущность, на которую смотрит игрок.
  - `<range>` — максимальная дистанция в блоках.
  - Если цель не найдена — каст отклоняется с причиной `NO_TARGET` (и сообщением из `messages.no_target`, если не задан `denyMessage`).

- `area r <radius> [center caster|target]` — цели = все подходящие сущности в радиусе.
  - `<radius>` — радиус в блоках.
  - `center caster` — центр области = позиция кастера.
  - `center target` — центр области = позиция найденной цели (как в `target <range>`).
  - Если `center target`, но цель не найдена — каст отклоняется как `NO_TARGET`.

## Примеры

```yml
- target: "self"
  actions:
    - "msg hi"

- target: "target 5"
  actions:
    - "dmg 4"

- target: "area r 4 center caster"
  actions:
    - "dmg 2"

- target: "area r 6 center target"
  actions:
    - "particles CRIT count 30 offset 0.4"
```
