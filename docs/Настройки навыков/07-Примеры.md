Готовый пример есть тут:
- `plugins/SkillsEngine/skills/example.yml`
- исходник: `src/main/resources/skills/example.yml`

## Команда + условия + удар по цели

Ниже примеры полностью готовых навыков. Все строки триггера/условий/действий лучше держать в кавычках (`"..."`) — так YAML проще переживает пробелы и спецсимволы.
```yml
id: smite
trigger: "cmd smite"
cooldown: 2s

conditions:
  - "perm skills.smite"
  - "lvl 10"
  - "world allow world,world_nether"

groups:
  - target: "target 5"
    actions:
      - "dmg 6.0"
      - "potion SLOW dur 3 amp 2"
      - "sound ENTITY_LIGHTNING_BOLT_THUNDER vol 0.8 pitch 1.2"
```

## Клик предметом + аура

```yml
id: aura
trigger: "rc BLAZE_ROD"
cooldown: 10s

conditions:
  - "world deny arena"
  - "item EMERALD 1 where inv"

groups:
  - target: "area r 4 center caster"
    actions:
      - "particles CRIT count 30 offset 0.4"
      - "dmg 2.0"
      - "fire 3"
```
