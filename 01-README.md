# SkillsEngine

SkillsEngine — минимальный движок навыков (skills) для Paper 1.21+.

Возможности (MVP):
- загрузка навыков из `plugins/SkillsEngine/skills/*.yml`
- триггеры: команда, клики мышью, удар по мобу, получение урона
- таргетинг: на себя / на цель / по области
- условия: permission, уровень, предметы, мир (allowed/denied)
- стоимость: уровни XP или предметы
- действия: урон/хил/эффекты/частицы/звук/команды/телепорт/и др.

Разделы:
- **Быстрый старт**: см. `quick-start.md`
- **Конфиг**: см. `config.md`
- **Формат навыков**: см. `Настройки навыков/01-Общее.md`
